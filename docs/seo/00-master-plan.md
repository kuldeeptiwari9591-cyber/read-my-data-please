# 00 — CrispPDF SEO Master Plan

Synthesized from files 01–08. This is the only file the implementation prompt has to read end-to-end.

---

## TL;DR

1. **India is the prize.** It's a 5–10× bigger market than the US for the exact same keywords, with weaker SERPs. Build India-first.
2. **The single biggest unbuilt opportunity** is the size-target compression niche: `/compress-pdf-to-100kb`, `-200kb`, `-500kb`, etc. — ~470K searches/mo combined, low KDI, India-driven. One template, 10 pages.
3. **Quick wins** live in low-KDI format pages — `png-to-pdf` (110K, KDI 49), `heic-to-pdf` (40.5K, KDI 39).
4. **CrispPDF already leads on `Question` + `Speakable` schema** (no major competitor has them) — keep pushing AEO advantage.
5. **The hardest head terms (`pdf editor` 246K, `edit pdf` 74K KDI 92) are not worth direct chasing**. Win them indirectly via long-tail clusters that link up to the category hub.
